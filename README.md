# Bitcasting

Projeto de pesquisa para prever a direção do preço do Bitcoin em múltiplos horizontes de tempo usando métricas on-chain, indicadores técnicos e análise de séries temporais. O repositório reúne notebooks que exploram dados históricos, métricas de blockchain e sinais técnicos para construir e avaliar modelos preditivos.

## Objetivos

- Investigar relações entre métricas de blockchain e a direção do preço do Bitcoin.
- Explorar indicadores técnicos clássicos e sua capacidade de antecipar movimentos.
- Comparar resultados em diferentes janelas temporais e combinações de sinais.

## Conteúdo do repositório

- `Block_Data.ipynb`: exploração e tratamento de dados on-chain.
- `historical_Data.ipynb`: análise de preços históricos e preparação de séries.
- `Techinical_crypto.ipynb`: construção e avaliação de indicadores técnicos.
- `Simulação.ipynb`: simulações e experimentos para avaliar estratégias.

## Como usar

1. Abra os notebooks em um ambiente Jupyter (JupyterLab, VS Code ou Google Colab).
2. Execute as células na ordem indicada em cada notebook.
3. Ajuste parâmetros, janelas e indicadores para comparar resultados.

## Requisitos sugeridos

- Python 3.10+
- Jupyter Notebook/Lab
- Bibliotecas comuns de análise de dados (pandas, numpy, matplotlib, scikit-learn)

## Status

Projeto em evolução. Novos experimentos e refinamentos de modelos podem ser adicionados conforme a pesquisa avança.
